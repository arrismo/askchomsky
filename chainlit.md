# Welcome to AskChomsky

Hi, I am the developer of AskChomsky.

AskChomsky is a retrieval-augmented chatbot over a Noam Chomsky corpus, built to make exploring ideas, sources, and arguments fast and transparent.

## Project Links

- GitHub repository: [arrismo/askchomsky](https://github.com/arrismo/askchomsky)
- Dataset used: [mmoise00/chomsky-corpus](https://huggingface.co/datasets/mmoise00/chomsky-corpus)

Try asking about media criticism, foreign policy, language, or corporate power, and request citations when you want source-grounded answers.
